COMMANDS                                                   *vim-buffr-commands*

                                                                       *:Buffr*

:Buffr {name}

  Open or create a buffer with {name}.
  Option 'name' is optional.

===============================================================================
OPTIONS                                                     *vim-buffr-options*

                                                             *g:buffr_position*

Default buffer position.

  Possible values: 'tab', 'top', 'bottom', 'left', 'right',
  'top-full', 'bottom-full', 'left-full', 'right-full'
  Default value: 'top'

===============================================================================
FUNCTIONS                                                 *vim-buffr-functions*

                                                  *buffr#open_or_create_buffer*

buffr#open_or_create_buffer({ 'name': {name}, 'position': {position} })

  Open or create a buffer with {name} at {position}.
  Options 'name' and 'position' are optional.

  Possible {position} values: 'tab', 'top', 'bottom', 'left', 'right',
  'top-full', 'bottom-full', 'left-full', 'right-full'
  Default {position} value: 'top'

                                                            *buffr#open_buffer*

buffr#open_buffer({ 'name': {name}, 'position': {position})

  Open a buffer with {name} at {position}.
  Option 'name' is required.

  Possible {position} values: 'tab', 'top', 'bottom', 'left', 'right',
  'top-full', 'bottom-full', 'left-full', 'right-full'
  Default {position} value: 'top'

                                                          *buffr#create_buffer*

buffr#create_buffer({ 'name': {name}, 'position': {position} })

  Create a new buffer with {name} at {position}.
  Options 'name' and 'position' are optional.

  Possible {position} values: 'tab', 'top', 'bottom', 'left', 'right',
  'top-full', 'bottom-full', 'left-full', 'right-full'
  Default {position} value: 'top'

CHANGELOG                                                 *vim-buffr-changelog*

1.0.0 (2017-09-03)~

  Code and documentation changes

0.3.0 (2016-03-26)~

  Changes
    * buffer name was changed to optional
    * function params were changed to named params
    * default position option was added

    * buffr#open function was separated to buffr#open_or_create_buffer,
      buffr#open_buffer, buffr#create_buffer

    * buffer name space escaping was added
    * open buffer silent

  First release
